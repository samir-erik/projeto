# 📰 Projeto: Site de Notícias

## 📌 Objetivo do Projeto 
 
 ## 🚀 Requisitos Não Funcionais

O sistema deve:

- 📱 Ser responsivo (desktop e mobile)
- ⚡ Possuir tempo de resposta de até 2 segundos
- 🔐 Garantir segurança básica (proteção da chave da API)
- 🌐 Estar disponível online via navegador
- 🎯 Interface simples e de fácil utilização

## 🛠️ Tecnologias Utilizadas

- **Front-end:** HTML, CSS, JavaScript  
- **Back-end:** Python (Flask)  
- **Banco de dados:** Não utilizado  
- **API externa:** GNews  
- **Hospedagem:** Render

- ## 🔄 Estrutura do Sistema

1. Usuário acessa o site pelo navegador  
2. Front-end envia requisições ao back-end  
3. Back-end consome a API GNews  
4. API retorna dados em JSON  
5. Back-end processa os dados  
6. Front-end exibe as notícias
