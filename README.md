📰 Projeto: Site de Notícias
📌 Objetivo do Projeto

Desenvolver um site de notícias com a finalidade de exibir informações atualizadas em tempo real, permitindo que os usuários busquem e filtrem conteúdos de forma simples e rápida.

📖 Descrição Geral

O projeto consiste no desenvolvimento de um sistema web que apresenta notícias de diversas categorias, utilizando uma API externa para obtenção de dados.

As notícias exibidas no site serão fornecidas pela GNews API, garantindo atualização constante das informações.

⚙️ Funcionalidades (Requisitos Funcionais)

O sistema deve permitir:

🔹 Exibir lista de notícias na página inicial
🔹 Buscar notícias por palavra-chave
🔹 Filtrar notícias por categoria
🔹 Exibir detalhes das notícias (título, descrição, imagem e link)
🔹 Exibir imagens relacionadas às notícias
🔹 Navegação entre diferentes notícias (paginação simples)
🚀 Requisitos Não Funcionais

O sistema deve:

📱 Ser responsivo (desktop e mobile)
⚡ Possuir tempo de resposta de até 2 segundos
🔐 Garantir segurança básica (proteção da chave da API)
🌐 Estar disponível online via navegador
🎯 Ter interface simples e de fácil utilização
🛠️ Tecnologias Utilizadas
Front-end: HTML, CSS, JavaScript
Back-end: Python (Flask)
Banco de dados: Não utilizado
API externa: GNews
Hospedagem: Render
🔄 Estrutura do Sistema

O funcionamento ocorre da seguinte forma:

O usuário acessa o site pelo navegador
O front-end envia requisições ao back-end
O back-end faz requisições à API GNews
A API retorna os dados em formato JSON
O back-end processa os dados
O front-end exibe as notícias ao usuário
🎯 Público-Alvo
Usuários interessados em acompanhar notícias atualizadas
Pessoas que buscam acesso rápido a informações de diversas categorias
⚠️ Limitações do Projeto

O projeto não incluirá:

❌ Sistema de cadastro ou login
❌ Criação própria de notícias
❌ Armazenamento em banco de dados
❌ Comentários ou interação social
📦 Entregáveis

Ao final do projeto será entregue:

📁 Código-fonte do sistema
🌍 Site funcional hospedado online
📄 Documentação básica
🎤 Apresentação do sistema
